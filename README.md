# Azure Self-Healing VM Project

This project demonstrates deployment of a Linux VM in Azure using Bicep with networking, security, and monitoring setup.

## Features
- Azure VM deployment using Bicep
- Virtual Network and NSG configuration
- Public IP exposure
- Nginx web server setup
- SSH-based access

## Tech Stack
- Azure Bicep
- Azure VM (Linux)
- Azure Networking (VNet, NSG)
- Nginx
